**Chehab Perfume Store**

A full stack e-commerce web application built with Node.js, Express, MongoDB, Docker, and PayPal Sandbox integration.

**1. Features:**

•	User registration and login.

•	Role based access control.

•	Product listing and search.

•	Shopping cart.

•	PayPal Sandbox payment integration.

•	Order history.

•	Docker with MongoDB database.

•	Secure backend payment verification.

•	Environment variable configuration.

**2. Technologies Used**

   **Backend**

   •	Node.js

   •	Express.js

   •	MongoDB

   •	Mongoose

   •	JWT

   •	bcryptjs

   •	Axios

   •	dotenv

   **Frontend**

   •	HTML5

   •	CSS3

   •	JavaScript

**3.  Installation Guide**
- Clone the repository



git clone https://github.com/Chippo90/App-Web-Development.git

cd perfume-store
- Install dependencies

npm install

- Run MongoDB with Docker

  docker run -d -p 27017:27017 --name perfume-mongo mongo:7
- Start the server

  npm run dev
